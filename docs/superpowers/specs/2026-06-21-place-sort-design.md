# 여행지 리뷰 정렬 설계

## 목표

여행지 탐색 목록의 카드에 실제 평균 별점과 리뷰 수를 표시하고, 사용자가 필요할 때만 추천순·리뷰 많은 순·평점 높은 순을 선택할 수 있게 한다.

## UI

- 목록 필터 아래에 `추천순`, `리뷰 많은 순`, `평점 높은 순` 버튼을 표시한다.
- 초기 상태에는 선택된 버튼이 없으며 기존 일반 순서를 사용한다.
- 선택한 버튼을 다시 클릭하면 정렬을 해제하고 일반 순서로 돌아간다.
- 필터 초기화 시 정렬도 해제한다.
- 장소 카드에 리뷰가 있으면 `★ 4.8 · 리뷰 23`을 표시한다.
- 리뷰가 없으면 `리뷰 없음`을 표시한다.

## 정렬

- 일반: 검색어가 있으면 기존 검색 관련도, 없으면 장소 ID 오름차순
- 추천순: 검색 관련도를 우선한 뒤 평점과 리뷰 수를 함께 반영한 추천 점수 내림차순
- 리뷰 많은 순: 검색 관련도를 우선한 뒤 리뷰 수 내림차순, 평균 평점 내림차순
- 평점 높은 순: 검색 관련도를 우선한 뒤 평균 평점 내림차순, 리뷰 수 내림차순
- 리뷰가 없는 장소는 모든 리뷰 기반 정렬에서 뒤에 배치한다.

추천 점수는 리뷰가 적은 5점 장소의 과대평가를 줄이기 위해 베이지안 보정 평균을 사용한다.
