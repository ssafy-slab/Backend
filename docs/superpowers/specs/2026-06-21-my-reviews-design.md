# 마이페이지 내 리뷰 탭 설계

## 목표

로그인 사용자가 마이페이지에서 자신이 작성한 장소 리뷰를 최신순으로 확인하고, 해당 여행지 상세 화면으로 이동하거나 리뷰를 삭제할 수 있도록 한다.

## 사용자 경험

- 마이페이지에 `내 리뷰` 탭을 추가한다.
- 탭 진입 시 작성한 리뷰 개수와 최신순 리뷰 목록을 표시한다.
- 각 카드에는 장소명, 장소 이미지, 별점, 리뷰 내용, 작성일을 표시한다.
- `여행지 보기`는 해당 장소 상세 화면으로 이동한다.
- `삭제`는 확인 후 리뷰 행을 실제 삭제하고 목록에서 제거한다.
- 리뷰 수정은 여행지 상세 화면에서 처리한다.
- 작성 리뷰가 없으면 빈 상태 안내를 표시한다.

## API

- `GET /api/users/me/reviews`
  - 로그인 필수
  - 현재 사용자의 활성 리뷰를 최신 작성순으로 반환
- 기존 `DELETE /api/places/{placeId}/reviews/me`를 삭제에 재사용한다.
- 기존 `GET /api/places/{placeId}`를 여행지 상세 이동에 사용한다.

## 응답 데이터

각 리뷰는 `reviewId`, `placeId`, `placeName`, `category`, `thumbnailImageUrl`, `rating`, `content`, `createdAt`, `updatedAt`을 포함한다.

## 범위 제외

- 마이페이지 인라인 수정
- 페이지네이션
- 정렬·필터 선택
